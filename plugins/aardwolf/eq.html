About

This plugin reads and parses invmon data from Aardwolf

Commands

eq

eq


usage: #bp.eq.eq [-n] [-c] [-s] [-h]

show equipment worn

optional arguments:
  -n, --noflags  don't show flags, default False
  -c, --score    show score, default False
  -s, --serial   show serial, default False
  -h, --help     show help

get


usage: #bp.eq.get [-h] [item] [otherargs [otherargs ...]]

get an item

positional arguments:
  item        the item to get
  otherargs   the rest of the args

optional arguments:
  -h, --help  show help

usage: #bp.eq.inv [-n] [-c] [-s] [-g] [-h] [container]

show inventory or a container

positional arguments:
  container      the container to see

optional arguments:
  -n, --noflags  don't show flags, default False
  -c, --score    show score, default False
  -s, --serial   show serial, default False
  -g, --nogroup  don't group items, default False
  -h, --help     show help
