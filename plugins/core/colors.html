About

This plugin handles colors

Color Codes: xterm 256 - make text color xterm 154 - make background color xterm 154

regular ansi: regular bold Red Green Yellow Blue Magenta Cyan White Reset @k

Commands

colors

example


usage: #bp.colors.example [-h]

show color examples

optional arguments:
  -h, --help  show help

show


usage: #bp.colors.show [-h]

show colors

optional arguments:
  -h, --help  show help

Base

api


usage: #bp.colors.api [-h] [api]

list functions in the api

positional arguments:
  api         api to get details of

optional arguments:
  -h, --help  show help

help


usage: #bp.colors.help [-a] [-c] [-h]

show help info for this plugin

optional arguments:
  -a, --api       show functions this plugin has in the api
  -c, --commands  show commands in this plugin
  -h, --help      show help

reset


usage: #bp.colors.reset [-h]

reset the plugin

optional arguments:
  -h, --help  show help

save


usage: #bp.colors.save [-h]

save the plugin state

optional arguments:
  -h, --help  show help

set


usage: #bp.colors.set [-h] [name] [value]

change a setting in the plugin

if there are no arguments or 'list' is the first argument then
it will list the settings for the plugin

positional arguments:
  name        the setting name
  value       the new value of the setting

optional arguments:
  -h, --help  show help

stats


usage: #bp.colors.stats [-h]

show plugin stats

optional arguments:
  -h, --help  show help
